# Build with KON

No-code On-chain App Framework

## Project structure

```
|--- packages
|------ api => API
|------ contracts => Smart Contracts
|------ shared => Shared ts lib
|------ shared-react => Shared react lib
|------ site => Shared react lib
|------ xmtp-agent => XMTP agent
|--- templates
|------ pwa => PWA template for app
```

## Dev

Install

```bash
pnpm install
```

Running example frontend

```bash
pnpm @pwa dev
```
